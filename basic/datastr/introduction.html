                            基本概念和术语

                            数据

                            数据是对客观事物的符号表示，所有输入到计算机中并被计算机处理的符号总和。

                            数据元素

                            数据元素是数据的基本单位，常常作为一个整体进行考虑。一个数据元素由若干个数据项组成。

                            数据对象

                            数据对象是性质相同的数据元素的集合。

                            数据结构

                            数据结构是数据元素相互之间存在的一种关系。可分为逻辑结构、存储结构和数据的运算。

                            数据类型

                            数据类型是一个值的集合和定义在其之上的一组操作的总称。

                            1. 原子类型：不可再分。

                            2. 结构类型：可以再分解成若干个数据类型。

                            3. 抽线数据类型：抽象数据组织和相关操作。

                            数据结构三要素

                            逻辑结构

                            逻辑结构是指数据元素之间的逻辑关系，可分为线性结构和非线性结构。

                            线性结构只有一种结构，表示一对一关系。

                            非线性结构：

                            1. 集合：元素之间无任何关系。

                            2. 树形结构，元素之间存在一对多关系。

                            3. 图状结构，元素之间存在多对多关系。

                            存储结构

                            存储结构是指数据结构在计算机中的表示，也称物理结构，不仅包含数据元素的表示，还有关系的表示。

                            1. 顺序结构：逻辑上相邻的元素在物理位置也相邻。

                            2. 链式结构：通过元素中存储地址的指针来表示元素之间的逻辑关系。

                            3. 索引结构：建立附加的索引表。

                            4. 散列存储：格局元素的关键字计算元素的存储地址。

                            算法和算法分析

                            算法

                            算法是对特定问题求解的描述，指令的有限序列，每一条指令包含一个或多个操作。算法有五个重要特性：

                            1. 有穷性：算法应该在有限步数之后结束，每一步也有有限时间。

                            2. 确定性：算法中的指令应当由明确含义，相同的输入只能得出相同的输出。

                            3. 可行性：算法中的操作通过基本运算执行有限次来实现。

                            4. 输入：一个算法有零个或多个输入。

                            5. 输出：一个算法有一个或多个输出。

                            算法和程序的区别

                            对于算法来说需要有穷，但是程序却未必一定需要，例如操作系统就是一个无限循环的系统。

                            好算法的要求：

                            1. 正确性：算法应当正常解决问题。

                            2. 可读性：算法应当由良好的可读性。

                            3. 健壮性：输入非法数据时，算法能适当处理。

                            4. 高效率和低存储需求：算法执行时间应短，占用空间少。

                            时间复杂度

                            语句在算法中倍重复执行的次数被称为时间复杂度。

                            算法中时间复杂度记为

                            T(n)=O(f(n))

                            T(n) 表示所有语法的频度之和，f(n) 表示算法中增长最快的项并忽略其系数，假设一个多次函数 f(n)=3n3+n2+n 一定是 3n3 增长最快，在 n 逐步增大的过程中，n2 和 n 可以忽略不记。而系数也同样进行忽略。

                            常见的渐近时间复杂度：

                            O(1)<O(log2⁡n)<O(n)<O(nlog2⁡n)<O(n2)<O(n3)<O(2n)<O(n!)<O(nn)

                            加法原则：

                            O(n)+O(m)=O(max(n,m))

                            乘法原则：

                            O(n)×O(m)=O(m×n)

                            空间复杂度

                            S(n) 表示空间复杂度，含义为所占存储空间的大小。
