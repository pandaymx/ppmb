## 定义

数组是一种数据结构，用于存储相同类型的一组元素。一般在内存中是连续进行分布的。通常通过索引进行访问。

```java
int[] arr = {1,2,3,4,5}
```

在编程语言中索引一般以0开始。

### 随机访问

通过索引进行访问，时间复杂度一般为 O(1)。

```java
System.out.println(arr[0]); 
```
