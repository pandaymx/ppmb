## 双列集合

双列集合 map 保存 key-value 键值对，key 值不可以重复，value 值可以重复。



### 常用方法

|方法名称|说明|
|---|---|
|put(key, value)|向 map 中添加元素|
|get(key)|获取指定 key 对应的 value|
|remove(key)|删除指定 key 对应的 value|
|containsKey(key)|判断 map 中是否包含指定 key|
|containsValue(value)|判断 map 中是否包含指定 value|
|size()|返回 map 中元素的个数|
|isEmpty()|判断 map 是否为空|
