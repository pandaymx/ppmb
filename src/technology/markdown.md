## 简介

Markdown是一种轻量级的标记语言，可用于在纯文本文档中添加格式化元素。

建议使用 VS Code 或者 Typora 来进行使用。

## 基本语法
### 标题

标题可以通过添加添加#号来添加。一级标题为1个#号，六级标题为6个#号。建议一级标题只使用一个。

建议在#号和后续的文本之间使用空格来进行分隔。

```md
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

演示效果：
> # 一级标题
> ## 二级标题
> ### 三级标题
> #### 四级标题
> ##### 五级标题
> ###### 六级标题

### 段落和换行

换行使用一个回车进行换行。另一种是使用HTML中的换行标签\<br>进行换行。

Markdown中的段落不建议使用中文中习惯的首行缩进，直接使用多个空白行进行分隔。

::: tip 更多
可以简单记作一个回车是换行，两个回车是段落。
:::

```md
第一段第一行
第二行

第二段
```

演示效果：
第一段第一行
第二行

第二段

### 特殊语法

主要有三种特殊语法，斜体、加粗和粗斜体。

| 名称  | 格式          |
|-----|-------------|
| 斜体  | \*文本*       |
| 粗体  | \*\*文本**    |
| 粗斜体 | \*\*\*文本*** |
|分割线|---|

```md
*斜体*，**粗体**，***粗斜体***

---
```

演示效果：

*斜体*，**粗体**，***粗斜体***

---  

### 引用

创建引用需要在段落前添加\>。

块引用也可以包含多个段落：需要在两个段落中添加一个空白的>符号。

```md
> 这是一个引用

> 你好
>
>第二段引用。

>嵌套语法
>
> > 第二层引用
> # 一个一级标题
```

演示效果：

> 这是一个引用

> 你好
>
>第二段引用。

> 嵌套语法
>
> > 第二层引用
> # 一个一级标题

### 列表

列表主要分为有序列表和无序列表，其中列表后可以嵌套其他元素，比如图片等，嵌套需要使用四个空格或一个制表符。

#### 有序列表

有序列表使用数字+.之后便可创建列表。

```md
1. 第一列
2. 第二列
3. 第三列

1. 第一列
    1. 第一个
    2. 你好
2. 第二列
3. 嵌套
    > 你好
```

演示效果：

1. 第一列
2. 第二列
3. 第三列

----
1. 第一列
    1. 第一个
    2. 你好
2. 第二列
3. 嵌套
    > 你好


#### 无序列表

无序列表使用-加空格即可创建。

```md
- 第一
- 第二
- 第三
```
演示效果：
- 第一
- 第二
- 第三

### 代码

将单词或短语包裹便可表示为代码。

```md
`代码`
```


演示效果：

`代码`

### 链接

链接文本在中括号内，地址在后面的括号中。前方加上感叹号则是图片。也可以是本地地址。
```md
[百度](https://www.baidu.com)


![图片](/logo.jpg)
```

演示效果：
[百度](https://www.baidu.com)


![图片](/logo.jpg)

### 转义字符

当某些字符需要按照原本的内容来显示时，需要使用转移字符\来对字符进行转移。

```md
\![百度](https://www.baidu.com)
```

演示效果：

\![百度](https://www.baidu.com)

## 扩展语法

### 表格

Markdown 使用三个连字符创建每列的标题，并使用\|来分隔每列。

```
| 测试| 有限|
| --- | ---|
|你好| 你好|
|我爱你|你爱我|
```

呈现输出如以下所示：

| 测试  | 有限  |
|-----|-----|
| 你好  | 你好  |
| 我爱你 | 你爱我 |

对齐方式：

```
| 测试 | 有限 |你好|
|:----|:----:|----:|
| 你好 | 你好 |你好|
|我爱你|你爱我|蜜雪冰城甜蜜蜜|
```
第一种表示左对齐，第二种表示居中对齐，第三种表示右对齐。



### 代码块

可以通过三个\`来创建代码块。其后可以添加一种语言，也可以不添加

\```python
\ print('hello world')
\```
效果如下所示：

```python
print('hello world')
```

### 列表

对需要定义的列表在前方添加冒号加空格。

```
第一列
: 我

第三人称
: it
: he
: she
```

### 删除线

使用两个~~来对文档添加删除线。

`~~你好~~`

演示效果：
~~你好~~

### 任务列表

通过一个-加空格加上[]来创建一个复选框，如果需要选中则在[]中添加一个x。

```
- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
```

演示效果：

- [x] Write the press release
- [ ] Update the website
- [ ] Contact the media
