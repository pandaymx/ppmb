## 简介

vi是Linux内建的文本编辑器，vim是其的增强，拥有程序编辑能力。

### 安装

```sh
apt install vim
```

![vim键盘图](https://www.runoob.com/wp-content/uploads/2015/10/vi-vim-cheat-sheet-sch.gif)

## 模式

### 命令模式

启动vim则进入命令模式，所有动作会识别为命令。

常见命令有：

- i：输入模式
- :：切换到底线命令模式

### 输入模式

输入模式和Windows中的大部分word操作类似，需要知道的是使用`Esc`返回命令模式。

### 底线命令模式

使用:好进入底线命令模式。

- w：保存文件
- q：退出vim
- q!：强制退出，不保存文件
