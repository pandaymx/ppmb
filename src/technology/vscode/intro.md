## 简介

Visual Studio Code是轻量级代码编辑器，内置有部分编程语言的支持，通过添加扩展来获得其他编程语言的支持。适用于Windows，macOS和Linux操作系统。

具有以下功能：

1. 智能代码补全
2. 简化调试
3. 代码导航和重构
4. 集成git等

## 安装

Visual Studio Code的安装可以直接在官网地址进行下载。
