## 基本介绍

环境变量是操作系统用来指定操作系统运行环境中的一些参数。

一般环境变量分为：

1. 用户变量
2. 系统变量

### PATH

Windows 中一般不区分大小写，但是 Linux 中需要区分大小写，输入命令后，系统会在 `PATH` 当中查找可执行文件。


## Windows 修改环境变量

打开 `此电脑` &rarr; `高级系统设置` &rarr; `环境变量` 在系统中添加需要的环境变量即可。
