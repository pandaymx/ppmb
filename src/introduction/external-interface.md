## USB 接口


### type

type A 是电脑中常用的接口，是如今使用最广泛的接口，大部分台式机都会有三个以上的 USB Type A 接口。

type B 不常用，但是 Micro-B 非常常用，曾经用在安卓机上。

type C 是未来的标准，如今虽然也很火，但是还没有达到 type A 的级别。

### USB 版本

USB 可以分为 2.0，3.0和未来的4.0。

其中 2.0 命名乱，但是也过去了。USB 3.0 其实就是 USB 3.1 Gen 1，也是 USB 3.1 Gen 1 5 Gbps。USB 3.1 是 USB 3.1 Gen 2 也是 USB 3.2 Gen 1×1。 

建议你直接看速率就好了，索性现在已经放弃了这样的命名方式，USB 3.0 更名为 USB 5 Gbps。以速率为命名方式。

一般主板上蓝色的是 USB 3.0 接口，其他颜色的是 USB 2.0 接口。

## PS/2

这是键鼠接口，现在的计算机基本没有或者只保留一个，如果有两个的话，记住一句话，绿鼠键紫就好。

## 网线接口

网线接口，现在一般是10M/100M/1000M自适应接口，因此你可以随便选择。但是需要注意的是，你插上也不一定有1000M速度，还需要你的路由器、网线全部支持才可以享受应有的速度。

使用超五类线就可以了。

## 音频接口

音频接口一般是以颜色进行标识，一般为3个或者6个。

一般上绿色是输出设备、红色是麦克风。


## 显示器接口

显示器接口如今以 DP 口和 HDMI 口为主，建议使用 DP 口，要快于 HDMI 口。

但是具体还是要参考一下速率，同代的话 DP 一般是大于 HDMI，不同代的话就无法保证。
