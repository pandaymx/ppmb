## 键盘键位

常见的键盘有 101 键、104 键和 87 键这几种键盘位数，87 键是削减了小键盘区来进行缩减的。


## 键盘区域

### 主键盘区

主键盘区中的键又分为三大类，即字母键、数字（符号）键和功能键。


CapsLock 键按下后将会锁定，输入字母会自动变为大写，再次按下后会变为小写，可以观察键盘上的指示灯是否亮着。

数字（符号）键，这些键一般上有两种形式，这种形式可以通过按下 `shift` 键来进行转换，从而输入第二个字符。

空格键（最长的键）用于输入空格、退格键用于删除一个字符，enter 键用于确认，具体的使用可以在一些网站进行观察一下。

tab 键、Ctrl 键、shift 键、win 键和 fn 键用于快捷键，需要和其他键配合使用。

### 功能键区

用于完成特定的功能。

Esc键：叫做取消键，位于键盘的左上角。在许多软件中它被定义为退出键，一般用做脱离当前操作或退出当前运行的软件。

从F1~F12是功能键，一般软件都是利用这些键来充当软件中的功能热键。例如用F1键寻求帮助。

PrintScreen：用于截图屏幕，不过用的不。

### 控制键区

1. PageUp：按-下这个键可以使屏幕向前翻一页
2. PageDown：按-下这个键可以使屏幕向后翻一页。
3. Home：按-下这个键可以使光标快速移动到本行的开始。
4. End：按-下这个键可以使光标快速移动到本科的末尾。
5. Insert：按-下这个键可以改变插入与改写状态。
6. Delete：删除光标所在位置上的字符。
7. 方向键：移动这四个键，可以使光标在屏幕内上下左右移动。

### 数字键区

1. NumLock（数字锁定键）∶锁定之后正常使用数字，否则会变为控制键中的功能。
2. 运算符号键：包括加（+）、减（-）、乘（*）、除（/）运算符。
3. Enter 键：叫小回车键，与主键盘上的回车键功能完全相同。

## 快捷键的使用

###  F 功能键


```tex
F1 显示程序或系统帮助内容
F2 重新命名所选项目
F5 刷新当前窗口
```
### ctrl 控制键

```tex
Ctrl + S 保存
Ctrl + W 关闭程序
Ctrl + X 剪切
Ctrl + C 复制
Ctrl + V 粘贴
Ctrl + A 全选
```

### alt 

```tex
Alt + F4 关闭当前程序
Alt + Tab 两个程序交换
```

### Windows 资源管理器

```tex 
Windows + D 显示桌面
Windows + E 打开资源管理器
Windows + R 运行
Windows + L 切换使用者
Windows + shift + s 截图
```
