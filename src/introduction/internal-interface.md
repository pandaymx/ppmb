## CPU

### CPU 插槽

CPU 插槽一般可以分为两种，一种是针脚在 CPU 上的，一种是针脚在主板上的，具体的可以查看官方的手册，这个不过多赘述。

### CPU 供电接口

CPU 供电接口可以分为 4pin、8pin、8+4pin、8+8pin，你甚至可以在 16pin 的主板上只插一个 4pin 的，但是建议是将它插满。

## 显卡

### 显卡插槽

显卡插槽一般只有速度上的区别，一般上装每一个主板的第一根显卡插槽即可。

### 显卡供电接口

显卡的供电接口一般是在显卡上面，和 CPU 类似，但不同的是 CPU 是 4+4 的组合，但是显卡是 6+2 的组合，也是建议直接插满。

## PCI 插槽

一般现在指的是 PCI-E，E 是扩展的意思，后续有 PCI-E $\times$[数字] 的形式，例如 PCI-E $\times 1$，数字只有 1、2、4、8、16。数字越大越好。16 是显卡再用，1 一般是其他硬件会使用，比如 wifi 模块。

## 内存插槽

不再过多赘述。

## 风扇接口

一般有一个系统和其他的风扇接口，一般四针的是主板可以自己控制，三针的主板不能控制，系统风扇有些主板不进行使用的话无法开机。

## 其他插槽

这些插槽一般是和机箱连接的。

1. USB 插槽：一般有多个，一般是 USB 2.0 的速度。
2. USB 3.0 插槽：一般是蓝色的，容易坏，要小心一点。
3. 条线插槽：一般有多个，根据位置和机箱的指示灯和开机键进行连接 。
